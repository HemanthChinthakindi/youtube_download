# 🎥 HHM4a YT Downloader

A sleek, fast, and cross-platform utility built to download YouTube videos and audio seamlessly. Available for both **Windows** and **Android**.

---

## 🚀 Features

* **Dual Platform Support:** Native application setups for Windows (`.exe`) and Android (`.apk`).
* **High-Quality Video:** Extract and save videos in multiple MP4 resolutions.
* **Audio Extraction:** Convert and download audio directly to MP3 format.
* **Sleek UI:** Clean, responsive, and user-friendly interaction layout.

---

## 📥 Installation

### Windows
1. Download **`HHm4a_YT_Downloader_setup.exe`** from this repository or the live website.
2. Run the installer package and follow the on-screen setup instructions.
3. Launch the application from your desktop shortcut or start menu.

### Android
1. Download **`HHM4a_YT_Downloader.apk`** to your mobile device.
2. Ensure "Install from Unknown Sources" is enabled in your device settings if prompted.
3. Open the file to complete the installation and launch the app.

---

## 🛠️ How to Use

1. **Launch** the application on your device.
2. **Paste** your desired YouTube video URL into the input address field.
3. Click **Fetch Info** to load available media streams and metadata.
4. **Select your format:**
   * Choose an **MP4** resolution to download the video file.
   * Choose **MP3** to extract and download only the audio track.
